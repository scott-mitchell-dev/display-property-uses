Display Property

this page illustrates the three most common values of the CSS display property . . .

  • block
  • inline
  • none

***block***

for the majority of html elements, the default display value is block . . .

html code:

<p class="red">paragraph #1</p> <p class="black">paragraph #2</p>

css code:

    p.black {
        color: black;
        font-size: 1.1rem;
        font-family: serif; }
    p.red {
        color: red;
        font-size: 1.1rem;
        font-family: serif; }

output:

inline paragraph #1

inline paragraph #2

note the word "block" is never specified in the code because it's the default.
But what if you want the two paragraphs next to each other? See below!!

***inline***

when you don't want vertical space, consider using display: inline; on your block elements

html code:

<p class="red inline">inline paragraph #1</p>
<p class="black inline">inline paragraph #2</p>

css code:

.inline { display: inline; }

output:

inline paragraph #1

inline paragraph #2

note that the inline property has to be used on both <p> elements

***none***

for responsive Web design, setting the display property to "none" is a critical tool

Large images (even Shake's) are not appreciated in most smart phone users' displays . . .
so how get rid of him for smart phones & other smaller displays?

ENTER THE MEDIA QUERY!!

HTML Code:

<img src="shake.png" class="disappear">

CSS Code:

@media only screen and (max-width: 767px) { .disappear {display: none; }}
